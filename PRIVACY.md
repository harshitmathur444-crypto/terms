# Falcon Modmail - Privacy Policy

## Data Collection
Falcon Modmail may collect and store:
- User IDs
- Server IDs
- Messages (for modmail/logging purposes)
- Server configuration data

## How Data is Used
- To provide modmail functionality
- To manage server features (AFK, attendance, logs)
- To improve bot performance

## Data Sharing
- No data is sold or shared with third parties.

## Data Storage
- Data is stored securely and only used for bot functionality.

## User Rights
- Server admins can request data removal by removing the bot.

## Changes
- This policy may be updated anytime.

## Contact
- Contact the developer for any concerns.
